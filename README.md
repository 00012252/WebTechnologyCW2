# NOTE TAKING APPLICATION

This application is intended to help to write your basic tasks 

#### Instructions

1. Downloading the source code

```bash
git clone 
```

or just download the \*.zip file

2. Install dependencies

```bash
npm install
```

3. Run the app

```bash
node app
```

### Dependencies List

- express
- pug
